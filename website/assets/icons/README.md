## Source

Use [Tabler Icons](https://tabler.io/icons) — free, open source, MIT licensed. Outline style only for consistency with brand.

## Format

- SVG preferred (scales perfectly, smallest file)
- 24x24 viewBox standard
- Single color (use `currentColor` in stroke so CSS can theme)
- Optimized via [SVGO](https://github.com/svg/svgo) before deployment
