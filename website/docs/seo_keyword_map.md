# SEO keyword map

Mapping of target keywords to pages. Used to keep on-page optimization aligned and avoid keyword cannibalization between pages.

**Methodology:** Each page targets ONE primary keyword (the search term it should win) and 2–4 secondary keywords (variations and modifiers). Geo-modifiers (Las Vegas, Henderson, Summerlin, etc.) appear naturally in headings, meta, and body — never stuffed.
