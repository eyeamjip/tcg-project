## Severity definitions

- **Critical** — blocks launch. Must fix before going live. (Broken form, missing critical content, security issue.)
- **High** — fix before launch. Affects conversion or major pages.
- **Medium** — fix this week. Visual bugs, minor content issues, accessibility issues.
- **Low** — nice-to-have. Log and address post-launch if time allows.
